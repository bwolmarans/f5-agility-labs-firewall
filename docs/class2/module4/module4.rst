Module 4: AFM Protocol Inspection IPS 
=====================================

In this lab you will explore the new Intrusion Prevention System feature in 13.1.0, which is called
Protocol Inspection. Protocol Inspection includes Compliance Checks and Signatures. This lab will introduce both, including a section on writing custom Signatures.
